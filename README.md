# 🛡️ Ethical Hacking Panel em Python

Projeto educacional desenvolvido em **Python** que simula um pequeno painel de ferramentas utilizadas em estudos de **segurança da informação e redes**.

O objetivo é reunir várias funcionalidades básicas em um **menu interativo estilo terminal hacker**.

---

## 📌 Funcionalidades

O painel possui algumas ferramentas simples:

* Scanner de portas
* Gerador de senhas seguras
* Teste de conectividade (ping)
* Interface em menu interativo

---

## ⚙️ Tecnologias Utilizadas

* Python 3
* Biblioteca `socket`
* Biblioteca `random`
* Biblioteca `string`
* Biblioteca `os`

## ▶️ Como Executar

Clone o repositório:

```
git clone https://github.com/SEU-USUARIO/ethical-hacking-panel-python.git
```

Entre na pasta do projeto:

```
cd ethical-hacking-panel-python
```

Execute o programa:

```
python security_panel.py
```
